A simple way to add a youtube video. Just put youtube::video_id:: and you'll see the video in the post. 

== Site ==
http://vladbabii.com/experimente/vb-simple-youtube

== Description ==

Example: youtube::CwyrGwM4a7M::. Just dont forget the space after the id.

== Installation ==

1. Download and unzip files
2. Upload the `vb-simple-youtube` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do i use it? =

Write a new post and put youtube::CwyrGwM4a7M:: in it. That's it. When you view the post you'll see the youtube video.
